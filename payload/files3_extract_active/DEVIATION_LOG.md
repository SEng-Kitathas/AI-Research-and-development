# Deviation Log
_Version: 2026-03-29_

All departures from pre-stated plans must be logged here BEFORE analysis begins.
Silent adoption of deviations is a process violation (R-18).
The original plan is preserved. This log records what changed and why.

---

## DEVIATION ENTRY TEMPLATE

```
DEVIATION ID:         [DEV-YYYYMMDD-NNN]
DATE:                 [timestamp]
EXPERIMENT ID:        [the pre-registered experiment this deviates from]
ORIGINAL PLAN STATED: [exact quote or reference from pre-registration log]
DEVIATION:            [what actually happened / what changed]
REASON:              [why the deviation occurred — honest account]
DIRECTION:            [did this deviation favor the hypothesis or oppose it?]
ANALYSIS IMPACT:      [does this change the interpretation of results?]
CORRECTIVE ACTION:    [what will be done to prevent recurrence, if applicable]
RETROACTIVE FLAG:     [are any prior conclusions now suspect? list them]
```
