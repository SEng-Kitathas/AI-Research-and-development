# TQ2 Multi-Path Latent Benchmark
_Version: 2026-03-29z_

## Goal
Test whether the latent state preserves multiple valid internal routes or collapses them into a single family groove.

## Setup
Each family has two valid internal route codes:
- Route A
- Route B

Both routes map to the same family outcome.

This benchmark measures:
- family accuracy
- route distinguishability under a route probe

## Important caveat
This is a synthetic route-diversity probe.
It does not claim to model full natural reasoning diversity.
It tests whether route identity survives inside the current family-state machinery.
