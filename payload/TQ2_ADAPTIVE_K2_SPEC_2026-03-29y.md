# Adaptive K2 Branch Spec
_Version: 2026-03-29y_

## Goal
Promote adaptive collapse/compression to a first-class variable on the live K2 branch.

## Working hypothesis
A latent geometric reasoner should not use one rigid explicit-collapse cadence for all states.
Difficulty / instability / confidence should modulate collapse frequency.

## Current proxy
Margin over the latent family state is used as the first adaptive signal.
