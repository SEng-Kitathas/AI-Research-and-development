# Generic Project-State Companion Archive

This archive is meant to sit beside the generic SOP archive.

## Use
1. Upload the generic SOP archive.
2. Upload this companion archive.
3. Duplicate/rename the templates into project-specific working files.
4. Start with:
   - Current State
   - Next Steps
   - Doctrine Snapshot
   - Revisit Ledger

## Intent
This gives a new project:
- load-first restart hygiene
- blank state/control files
- experiment/report templates
- zero-loss extraction starter forms
- archive index/checklist

It is deliberately project-agnostic.
