# Weight Substrate Basis Document
_Version: 2026-03-29b_

## Purpose
This basis supersedes 2026-03-29a after the substrate probe, the formal TQ2 object grammar, and the benchmark harness specification.

The program is now officially in **instrumented substrate research** mode.

---

## 1. What changed
Three things are now explicit:
1. TQ2 remains the build-favored substrate branch.
2. Whole-plane and blockwise are sibling RB-2 expression branches.
3. The project now has an instrument layer: object grammar + benchmark harness.

---

## 2. Canonical substrate position
The shared substrate remains:
- discrete
- low-bit
- hardware-sympathetic
- audit-friendly
- built from structured local objects
- evaluated under cheap closed transforms

TQ2 remains the current best lab bench for this work.

---

## 3. Canonical RB-2 split
RB-2 is now explicitly split into:
- RB-2A — whole-plane transforms
- RB-2B — blockwise transforms
- RB-2C — hybrid transforms

No implementation may flatten this distinction away without evidence.

## 5. Canonical doctrine

### D-1 — TQ2 first
The project should implement and benchmark TQ2 before deep exotic expansion.

### D-2 — Branches must fight
Expression branches compete under shared tasks and metrics.

### D-3 — Forge is translation pressure
RB-5 exists to test what survives translation into native substrate form.

### D-4 — Instrument before mythology
The grammar and harness outrank seductive prose.

### D-5 — Kill weak branches fast
If a branch cannot win on its home task, kill or demote it.

---

## 6. Immediate next sequence
1. build slow reference implementations for RB-2A, RB-2B, RB-2C
2. add mixed-task benchmark (global + local)
3. test whether RB-2C actually earns promotion
4. then connect the smallest viable Forge-to-TQ2 translation test

## 7. Standing interpretation
The project is no longer best described as “finding the one true weight format.”
It is best described as:

> discovering the strongest realizable discrete geometric weight substrate by forcing sibling branches to compete under a shared instrument layer while treating ingestion as a first-class bridge.
