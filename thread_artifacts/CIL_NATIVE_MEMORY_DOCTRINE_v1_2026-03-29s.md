# CIL-Native Memory Doctrine v1
_Version: 2026-03-29s_

## Thesis
CIL is the persistent exact ledger core of the memory architecture.
It is not the whole cognition stack.
It is the durable substrate around which the live, consolidating, geometric, replaying intelligence is built.

## Native doctrine

### 1. Dual-plane memory contract
The architecture is governed by two cooperating planes:
- Exact Plane: reversible fidelity, canonicalization, provenance, exact residuals, auditability
- Representational Plane: clustering, family formation, geometric topology, compressed retrieval, memory shaping

Both planes exist throughout the stack, with different priority by layer.

### 2. Stable object contract
All memory objects should converge toward a stable Radical memory object contract that can live in:
- session
- episode
- semantic
- archive

The lifecycle changes.
The representation contract does not.

### 3. CIL as substrate, not sidecar
The following layers should be treated as native around CIL:
- Session Buffer above CIL
- CIL Episodic Layer
- CIL Semantic Layer
- CIL Vector / Neighborhood Layer
- CIL Integrity Layer
- Consolidation Engine around CIL
- Replay / Sleep layer
- Retrieval Compiler above CIL
- Archive / Audit layer

### 4. Consolidation engine is center of gravity
Storage becomes memory only when witnesses are:
- radicalized
- clustered
- contradicted
- replayed
- promoted
- superseded

### 5. Geometry is memory topology
Hilbert / SFC should be treated not as decorative compression order, but as:
- replay topology
- family locality
- retrieval clustering
- recurrence vs novelty exposure
- compaction ordering

### 6. Radical doctrine
Radicals have two jobs:
- Exact Plane: reversible factors
- Representational Plane: family anchors

Promotion ladder:
- Seed radicals
- Mined radicals
- Apex radicals

Apex promotion should require:
- residual reduction
- normalization / isomorphism stability
- recurrence
- contradiction survival
- provenance quality

### 7. Retrieval doctrine
Do not dump raw ledger contents into working context.
Compile retrieval products:
- working packet
- evidence packet
- contradiction packet
- compressed family packet

### 8. Replay doctrine
Replay is mandatory.
Without replay, memory calcifies.
Without archive, superseded truth disappears.
Without contradiction memory, promotion becomes propaganda.
